Huong dan chay ung dung CRM-Mini ban MySQL XAMPP

B1: Tai va cai Node.js qua link https://nodejs.org/en/download
B2: Tai va cai XAMPP qua link https://www.apachefriends.org/download.html
B3: Mo XAMPP Control Panel va bam Start MySQL
B4: Giai nen Folder CRM-Mini-M-XAMPP
B5: Chay start-crm-mini.bat

Launcher se tu:
- Cai thu vien neu chua co
- Tao database MySQL ten crm_mini_xampp neu chua co
- Tao bang va nap du lieu demo
- Mo ung dung tai http://localhost:5173
